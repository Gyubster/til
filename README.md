# TIL (Today I Learned)

백엔드 엔지니어 심규보의 일일 학습 기록입니다.

## 이 레포에 대해

4년차 백엔드 엔지니어로서, 단순히 업무를 반복하는 것이 아니라 매일 의도적으로 성장하고 그 과정을 기록하기 위해 이 레포를 운영합니다.

매일 학습한 내용을 카테고리별로 정리하고, 단순 개념 정리가 아닌 **실제로 적용하고 느낀 점**을 중심으로 작성합니다. 작은 것이라도 매일 하나씩 쌓아가는 것이 목표입니다.

### 운영 방식

1. **매일 학습/실습한 내용을 기록** — 알고리즘 문제 풀이, 시스템 설계 학습, 실무에서 배운 점 등
2. **카테고리별 분류** — 어떤 영역에 시간을 투자하고 있는지 한눈에 파악
3. **꾸준함이 핵심** — 완벽한 글보다 매일 기록하는 습관을 우선
