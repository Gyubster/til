## 심규보 / Backend Engineer

개발은 고객 가치로 이어질 때 의미가 있다고 믿습니다.
최신 기술보다 문제의 본질을 빠르게 파악하고 올바른 방향으로 끝까지 실행하는 능력이 더 중요하다고 생각합니다.

4년간 B2B 핀테크에서 production 시스템을 책임지고 운영해 왔습니다.
실패가 곧 사고로 이어지는 영역에서 시스템을 끝까지 책임지는 방식으로 일합니다.

---

### 일하는 방식

- **임팩트 있는 문제를 스스로 정의하고 끝까지 책임집니다.** 일 약 6만 건 규모의 대출 가심사 파이프라인을 단독으로 설계·운영하며, 운영 초기 단계에서 데이터 불일치 구조를 직접 발굴해 수정했습니다.
- **장애는 증상이 아니라 구조로 해결합니다.** 반복되는 장애의 원인을 시스템 단위까지 파고들어, 같은 유형의 장애가 재발하지 않도록 만듭니다. 한 번의 회고가 팀 표준이 되도록 정착시킵니다.
- **AI 도구를 일상의 핵심으로 사용합니다.** Claude Code · Anthropic API를 1년 이상 dev workflow의 핵심으로 사용해왔고, 사내 자동화 에이전트를 설계·구현하고 있습니다.
